# Image Library

All site images (non-logo) live in this folder so they can be referenced consistently.

Naming convention (kebab-case, ASCII only):

<page-or-section>-<subject>-<action-or-angle>-<variant>.<ext>

Rules:
- Use lowercase letters, numbers, and hyphens only
- Keep names concise but descriptive
- Use a short variant tag when there are multiple versions (e.g., v1, v2, wide, square)
- Keep the file extension consistent with the actual format (jpg, png, webp)

Examples:
- home-hero-veteran-handshake-wide.webp
- about-hero-veteran-family-v1.jpg
- programs-therapy-dog-closeup.webp
- donate-callout-volunteer-smile-square.png

Optional subfolders (use only when the main folder gets crowded):
- heroes/
- about/
- programs/
- events/
- campaigns/
- misc/
